# Step 02 — Vacuum MD (真空中の MD)

> 必要な前知識: Step 01 完了。詳細は [GLOSSARY](../../../GLOSSARY.md)

## このステップで何をやるか

DNA 分子を **何もない真空中** (= 水もイオンもなし) で MD します。さらに、**カットオフを 12 Å にした版** と **無限大 (= カットオフなし) 版** の 2 通りで走らせて結果を比較します。

具体的なアウトプット:

- `min.ncrst` — 最小化後の座標
- `md_12Acut.nc`, `md_12Acut.out` — 12 Å カットオフ版 MD のトラジェクトリとログ
- `md_nocut.nc`, `md_nocut.out` — 無限大カットオフ版 MD のトラジェクトリとログ

---

## 1. なぜ「真空 MD」をやるのか

### 普通はやらない

**生体内の現実**は水溶液の中なので、**実用的な MD は水中で行います**。真空 MD はほとんど使われません:

- DNA のリン酸基の負電荷 (-18) が **水に隠れず剥き出し** になる
- 静電反発で構造が崩壊しやすい
- 実験で観察される DNA の挙動とはズレる

### でも教育用には超有用

それでもチュートリアルで真空 MD をやるのは:

| 真空 MD の学びポイント |
|---|
| **カットオフの影響** が極端に出るので、その重要性を体感できる |
| **計算が速い** (水分子がないので原子数が DNA の 640 個だけ) |
| **何が起きているか観察しやすい** (水のノイズなし) |
| **GB や明示溶媒との比較** で「**溶媒効果とは何か**」が浮き彫りになる |

特に最後のポイントが重要。Step 03 (GB) と Step 04 (明示溶媒) と並べて見ることで、「**水を入れるとなぜ DNA が安定するのか**」が劇的に分かります。

## 2. カットオフ (cutoff) — 物理的に何が起きているのか

### 真面目に計算するとどれくらい重いか

原子が N 個ある系で、すべての原子ペア (i, j) の相互作用を計算すると、**N×N/2 = O(N²)** ペア。

- 10 bp DNA: 約 640 原子 → 約 200,000 ペア
- 水中 DNA: 約 10,000 原子 → 約 5,000 万ペア
- タンパク質: 数万原子 → **数億ペア**

この計算を MD のステップごと (= 100 万回くらい) やるとしんどい。

### カットオフの発想

「**遠く離れた原子同士は相互作用が小さいから、無視しちゃおう**」というアイデアが **カットオフ**。距離 r が cutoff (例: 12 Å) を超える原子ペアは、力 F = 0 とみなして計算しません:

```
原子 A
  │
  │  距離 r が cut 以下:  まじめに F = -dE/dr を計算
  │  距離 r が cut 超え:  F = 0 とみなす (= 無視)
  │
原子 B
```

これで O(N²) を **おおよそ O(N)** に落とせます。原子 1 個が「cut 半径内に持つ近所」は系のサイズに依らず一定数だから。

### Lennard-Jones は cutoff OK、でも静電は…?

| 相互作用 | 距離依存 | cutoff の妥当性 |
|---|---|---|
| Lennard-Jones (van der Waals) | r⁻⁶ で減衰 | **8〜10 Å で安全** (遠くは無視できるほど小さい) |
| 静電 (Coulomb) | r⁻¹ で減衰 | **長距離まで効く** (cutoff で切ると誤差大) |

つまり **静電だけは特別扱いが要る**。

### PME (Particle Mesh Ewald) という救済

水中 MD では `PME` という賢いトリックで「**遠距離の静電を別途逆空間 FFT で計算**」して、cutoff の中だけ直接計算でも正確に長距離を含めます。これが現代 MD の標準。詳しくは [GLOSSARY § PME](../../../GLOSSARY.md#particle-mesh-ewald-pme)。

しかし **PME には周期境界条件が必要**。真空 (= 周期なし) の系では使えない。なので **真空では cutoff の影響がモロに出る**。

## 3. 12 Å vs 無限大 cutoff — 何が違うのか

このステップでは同じ DNA に対して 2 通りの cutoff で MD を走らせて比較します。

### `cut = 12.0`

12 Å を超える原子ペアの相互作用は **無視**:

```
DNA の原子 (n=640)
  │ 12 Å 以内のペアだけ計算 (各原子 ~50 個) → 約 32,000 ペアの計算
  │ 12 Å を超えるペアは「ない」とする → 無視される静電あり
```

利点: 速い  
欠点: **遠くのリン酸 -1 同士の反発が抜ける** → DNA が現実より安定に見える可能性 (or 不安定になる可能性、状況による)

### `cut = 999`

事実上 **すべてのペアを計算** (cutoff > 系のサイズ):

```
DNA の原子 (n=640)
  │ 全ペア計算 → 約 200,000 ペアの計算 (6 倍重い)
  │ 静電も全部計算される
```

利点: **正確** (cutoff 由来の誤差なし)  
欠点: 重い (大規模系では非現実的)

### このステップの教育的ポイント

「**12 Å cutoff**」と「**無限大 cutoff**」で同じ DNA を走らせて軌跡を比べると、

- カットオフによって **DNA backbone の RMSD の振る舞いが大きく変わる**
- 真空 MD では本来含めるべき遠距離静電が cutoff で切られると挙動がおかしくなる
- → **「真空 MD では cutoff を切ってはいけない」** ことが体感的に分かる

実際、**現実の MD で `cut=999` は普通やらない**。なぜなら大きい系だと O(N²) で重すぎるから。**水中 MD で PME を使うのが現代の標準** という話に繋がります。

## 4. 入力ファイルの解説

### `min.in` (最小化)

NAB から作った直後の DNA 構造には水素原子の最適化等が必要なので、まず最小化:

| パラメータ | 値 | 意味 |
|---|---|---|
| `imin` | `1` | 最小化 |
| `maxcyc` | `500` | 500 サイクル |
| `ncyc` | `250` | 前半 250 を steepest descent |
| `ntb` | `0` | **周期境界なし** ← 章 01 と違う重要点 |
| `igb` | `0` | **暗黙溶媒なし** (= 純粋に真空) |
| `cut` | `12` | カットオフ 12 Å |

### `md_12Acut.in` / `md_nocut.in` (MD 本体)

両方とも:
- **100 ps の MD** (`nstlim=100000 × dt=0.001`)
- `dt=0.001` (1 fs) ← 章 01 の 2 fs より短い
- **300 K で開始** (`tempi=300`) ← 章 01 と違って 0 K から加熱しない
- Langevin thermostat (`ntt=3`)

違いは `cut` の値だけ:
- `md_12Acut.in`: `cut=12.0`
- `md_nocut.in`: `cut=999`

### 章 01 との重要な違い

| 項目 | 章 01 (アラニン) | 章 02 (真空 DNA) |
|---|---|---|
| `dt` | 0.002 (2 fs) | **0.001 (1 fs)** |
| SHAKE (`ntc`/`ntf`) | 2 (有効) | **設定なし (デフォルト 1, 1 = 無効)** |
| 加熱ステップ | 0→300K で 20 ps | **なし、いきなり 300 K** |
| `ntb` | 1 か 2 (周期) | **0 (周期なし)** |

#### なぜ dt=1 fs なのか

X-H 結合の振動周期は約 10 fs。Nyquist の定理的に、これを安定に積分するには `dt < 0.5 fs` くらい必要。SHAKE で結合長を固定すれば、振動が「凍結」されるので 2 fs まで伸ばせる (章 01)。

ところが今回は **SHAKE を使っていない** (`ntc=ntf=1` がデフォルト)。なので:

```
dt = 2 fs → X-H 振動を捉えきれず数値発散
dt = 1 fs → ギリギリ安全
```

「**SHAKE 使わないなら dt=1 fs**」は MD の鉄則です。

#### なぜ加熱ステップがないのか

最小化済み + 真空 → 「**ぶつかる相手がいない**」ので、いきなり 300 K の運動エネルギーを与えても局所ホットスポットが起こりにくい。

水中だと「水分子が DNA に衝突して局所的に過熱」が起きるが、真空にはそれがない。

#### なぜ周期境界なしなのか

真空中に **DNA が 1 個ポツンとある状況** を再現したい。周期境界にすると「無限に並んだ DNA の格子」をシミュレートしたことになり、隣の像と相互作用してしまう。

## 5. 実行してみる

```bash
cd workspace
mkdir -p 02_vacuum_md && cd 02_vacuum_md

# 入力ファイルを 3 つコピー
cp ../../solutions/02_vacuum_md/*.in .

PARM=../01_setup/polyAT_vac.prmtop
INIT=../01_setup/polyAT_vac.rst7

# (a) 最小化
sander -O \
    -i min.in \
    -o min.out \
    -p $PARM -c $INIT -r min.ncrst

# (b) 12 Å cutoff の MD
sander -O \
    -i md_12Acut.in \
    -o md_12Acut.out \
    -p $PARM -c min.ncrst -r md_12Acut.ncrst -x md_12Acut.nc

# (c) 無限大 cutoff の MD
sander -O \
    -i md_nocut.in \
    -o md_nocut.out \
    -p $PARM -c min.ncrst -r md_nocut.ncrst -x md_nocut.nc
```

100 ps の vacuum MD は CPU で **5〜15 分** くらい。`cut=999` 版は `cut=12` 版より少し重いです (O(N²) 計算)。

## 6. 出力の見方

```
02_vacuum_md/
├── min.in / min.out / min.ncrst
├── md_12Acut.in / .out / .ncrst / .nc      ← 12 Å cutoff
└── md_nocut.in / .out / .ncrst / .nc       ← 無限大 cutoff
```

### `out` ファイルで確認するポイント

```bash
# 温度の推移を比較
grep "TEMP(K)" md_12Acut.out | awk '{print $4, $9}' | head -10
grep "TEMP(K)" md_nocut.out  | awk '{print $4, $9}' | head -10
```

両方とも **300 ± 数 K** で振動するはず (Langevin が温度制御している証拠)。  
温度がおかしいときは数値発散の前兆。

```bash
# Etot のドリフト確認
grep "Etot" md_12Acut.out | head -10
```

`Etot` (全エネルギー) が大きく変動していたら不安定の兆候。

### 期待される観察 (Step 05 で詳細比較)

| | 12 Å cutoff | 無限大 cutoff |
|---|---|---|
| 数十 ps での挙動 | 構造が**崩壊** (RMSD が 5 Å 以上に上昇) | もう少し持つが**不安定** (3-5 Å) |
| 解釈 | 遠距離静電が抜けて電荷バランスが歪んだ | 全静電を計算しても、水なしでは DNA は不安定 |

これが「**真空 MD は普通やらない**」理由を実体験できる仕掛けです。

## 7. つまずきポイント

| 症状 | 原因と対処 |
|---|---|
| `vlimit exceeded` で MD が爆発する | 真空 + 静電大 + 加熱なしの組み合わせは爆発しやすい。`min.in` の `maxcyc` を増やす (1000) |
| `cut=999` で MD が遅すぎる | 系が大きいと不可避。10 bp DNA なら問題なし、もっと大きい系では `cut=999` は使わない |
| 温度が暴れる | `gamma_ln` を上げる (1.0 → 2.0) で抑制力を強める |
| `ntb` の値で混乱 | `ntb=0` (周期なし) を真空 / GB で使う、`ntb=1` (NVT) や `ntb=2` (NPT) を明示溶媒で |
| ファイル名が一致しない | sander コマンドラインの `-c` (入力座標) と `-r` (出力座標) を混同しやすい。`-r` 側が次のステップの入力になる |

## 8. もう少し知りたい — cutoff の歴史

### 1980s〜1990s: cutoff だらけの時代

PME 以前は全 MD で cutoff を使うしかなかった。**8〜12 Å** が標準。タンパク質研究の黎明期はこれで進んでいた。

### 1990s: PME の登場

Tom Darden らが Particle Mesh Ewald を MD に応用 (1993)。これで「**短いカットオフ + 長距離 PME**」のハイブリッドが可能に。**水中 MD の精度が劇的に向上**した。

### 現在

- **水中 MD**: cutoff 8 Å + PME (= 章 01 の標準)
- **GB MD**: cutoff 12-16 Å (PME 不要、GB 自体が長距離処理)
- **真空 MD**: cutoff 大きめ or 無限大 (PME 使えないので妥協)

---

## 9. 用語ミニ辞典

- **真空 MD**: 水・イオンなしの MD。教育用以外で滅多に使われない
- **cutoff**: 「これ以上遠い原子間相互作用は無視する」距離
- **PME (Particle Mesh Ewald)**: 周期境界下での長距離静電を効率的に計算する手法。詳細 [GLOSSARY](../../../GLOSSARY.md#particle-mesh-ewald-pme)
- **SHAKE**: 結合長を固定して dt を伸ばすアルゴリズム。詳細 [GLOSSARY](../../../GLOSSARY.md#shake)
- **Etot**: 全エネルギー (運動 + ポテンシャル)。MD 中の保存性の指標
