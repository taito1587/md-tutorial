# Step 04 — Explicit Solvent MD (明示溶媒で本格的な 4 段階 MD)

## このステップで何をやるか

DNA を **TIP3P 水分子で囲み、Na+ で中和した「リアルな系」** で MD します。**4 段階に分けた段階的な平衡化** が必要で、章 01 (アラニン) より一歩進んだ手順です。

具体的なアウトプット:

- `min1.ncrst` — 段階 1: 溶媒だけ最小化 (DNA は固定)
- `min2.ncrst` — 段階 2: 全系最小化 (制約解除)
- `heat.ncrst`, `heat.nc`, `heat.out` — 段階 3: 0→300 K 加熱
- `prod.ncrst`, `prod.nc`, `prod.out` — 段階 4: 本番 NPT MD

トポロジーは Step 01 で作った **`polyAT_wat.prmtop`** (水 + イオン入り) を使います。

---

## 1. なぜ 4 段階に分けるのか

章 01 (アラニン) は `1. 最小化 → 2. 加熱 → 3. 本番` の 3 段階でしたが、ここでは:

```
段階 1: 溶媒のみ最小化 (DNA を 500 kcal/mol/Å² で固定)
   ↓
段階 2: 全系最小化 (制約解除)
   ↓
段階 3: 0→300 K 加熱 (NVT, DNA に弱い制約 10 kcal/mol/Å²)
   ↓
段階 4: 本番 NPT MD (制約なし)
```

の 4 段階になります。

### 直感的な理由

```
                                 状態        DNA 構造        溶媒の状態
─────────────────────────────────────────────────────────────────────
tleap 直後 (Step 01)              ガサガサ   理想的 (NAB 由来)  乱雑 (詰め込んだだけ)
                                                              ↓
段階 1 終了 (溶媒だけ最小化)     落ち着く   理想的 (固定)    DNA 周りに整列
                                                              ↓
段階 2 終了 (全系最小化)         安定        水と協調         DNA とフィット
                                                              ↓
段階 3 終了 (NVT 加熱)           300 K     ほぼ動かない (拘束) 300 K で熱運動
                                                              ↓
段階 4 進行中 (NPT 本番)         300 K     自由に揺らぐ      自然な液体
```

各段階で「**1 つだけ自由度を増やす**」のがコツ。一気に全自由度を解くと系が壊れます。

### なぜ DNA を最初固定するのか

`tleap` の `solvateoct` で水を箱に詰め込むと:

- 水分子はランダムな配向で並んでいる (= 平衡状態ではない)
- 一部は DNA とぶつかりかけている

この「**初期構造の悪い水分子の歪み**」を解消したいのですが、DNA も同時に動かすと **DNA が水の歪みに引きずられて変形**してしまいます。これは物理的に正しい挙動ではありません。

そこで **DNA を強い拘束 (バネ定数 500 kcal/mol/Å²) で固定** し、**水と Na+ だけを動かして** 周辺をリラックスさせます。これが段階 1。

その後で全自由度を解いて (段階 2)、ゆっくり加熱 (段階 3) → 本番 (段階 4) と進めます。

### この段階分けは「重い系の MD の標準作法」

DNA 以外でも、**膜タンパク質、タンパク質-リガンド複合体、DNA-タンパク質複合体** など、複数の構成要素が絡む系では同じパターン (溶媒先 → 重要部分後) が定石。覚えておくと一生使えます。

## 2. 拘束 (restraint) の仕組み

### `ntr=1` と `restraintmask`

`ntr=1` で位置拘束モードを有効にし、AMBER 入力ファイルの `&cntrl` ブロック直後に **拘束ブロック** を書きます:

```
ground state restraint title
500.0
RES 1 20
END
END
```

| 行 | 意味 |
|---|---|
| `Hold the DNA fixed` | コメント (1 行任意) |
| `500.0` | バネ定数 (kcal/mol/Å²) |
| `RES 1 20` | 拘束する残基番号 (1〜20、つまり全 DNA) |
| `END` | グループの終わり |
| `END` | 全拘束ブロックの終わり |

これで「**残基 1〜20 (= DNA の両鎖)** をバネ定数 500 で初期位置に固定する」となります。

### バネ定数の意味

```
拘束エネルギー = (k/2) × (現在の位置 - 初期位置)²
```

これは Hooke の法則 (バネ) と同じ式。`k` がバネ定数。

- **500 kcal/mol/Å²** は **かなり強い**バネ → ほぼ動かない
  - 1 Å ズレるだけで 250 kcal/mol のエネルギー罰
  - 室温の熱エネルギー RT ≈ 0.6 kcal/mol と比べると桁違いに強い
- **10 kcal/mol/Å²** は **弱め** → ある程度ゆらぐ
  - 1 Å ズレで 5 kcal/mol → 数倍 RT 程度

加熱中 (段階 3) は強くしすぎるとサーモスタットがうまく働かないので、**10** に下げています。

### 制約を使うシーン

DNA だけでなく:

- **タンパク質-リガンド複合体**: 複合体構造を最小化中に動かさないため
- **膜タンパク質**: 脂質二重膜が再配置するまでタンパク質を固定
- **遷移状態**: 反応経路上の特定の構造を維持
- **フリーエネルギー計算**: 反応座標を漸進的に動かす

「制約付き MD」は応用範囲がとても広いテクニック。

## 3. 各段階の入力ファイル

### Stage 1: `min1.in` — 溶媒だけ最小化

```
Stage 1: minimize SOLVENT (water + ions) only, hold DNA fixed
 &cntrl
  imin   = 1, maxcyc = 1000, ncyc = 500,
  ntb    = 1,
  ntr    = 1,
  cut    = 10.0
 /
Hold the DNA fixed
500.0
RES 1 20
END
END
```

### Stage 2: `min2.in` — 全系最小化 (制約なし)

```
 &cntrl
  imin   = 1, maxcyc = 2500, ncyc = 1000,
  ntb    = 1,
  ntr    = 0,
  cut    = 10.0
 /
```

### Stage 3: `heat.in` — 0→300 K 加熱 (NVT、DNA に弱い拘束)

ポイント:
- `tempi=0.0, temp0=300.0` で温度を上げる
- `ntr=1` で **弱い (10 kcal/mol/Å²)** 拘束を維持
- `ntc=2, ntf=2` で **SHAKE 有効** → `dt=0.002` (2 fs) が使える
- **NVT** (`ntb=1, ntp=0`)

### Stage 4: `prod.in` — NPT 本番 MD

```
 &cntrl
  imin     = 0,
  irest    = 1,        ! 加熱を継承
  ntx      = 7,        ! 座標+速度+箱情報を読む
  ntb      = 2,        ! NPT
  ntp      = 1,
  pres0    = 1.0,
  taup     = 2.0,
  ...
  ntr      = 0,        ! 拘束なし
  ...
  nstlim   = 50000,    ! 100 ps
 /
```

## 4. 章 01 (アラニン) との違い

| 項目 | 章 01 | 章 02 (DNA 明示溶媒) |
|---|---|---|
| 最小化 | 1 段 | **2 段 (溶媒先 → 全系)** |
| 加熱 | 制約なし | **DNA に弱い制約** |
| 本番の長さ | 10 ns | 100 ps (公式そのまま、伸ばすこと推奨) |
| `pres0` | デフォルト (1 atm) | **明示的に 1.0** |
| `barostat` | 1 (Berendsen) | デフォルト (1) |

「制約付きで段階的に立ち上げる」のが大きな違い。

## 5. 実行してみる

```bash
cd workspace
mkdir -p 04_explicit_solvent && cd 04_explicit_solvent

cp ../../solutions/04_explicit_solvent/*.in .

PARM=../01_setup/polyAT_wat.prmtop
INIT=../01_setup/polyAT_wat.rst7

# Stage 1: 溶媒だけ最小化 (DNA 拘束)
sander -O \
    -i min1.in -o min1.out \
    -p $PARM -c $INIT -r min1.ncrst \
    -ref $INIT     # ★ 拘束の参照位置を指定 (-ref フラグ)

# Stage 2: 全系最小化
sander -O \
    -i min2.in -o min2.out \
    -p $PARM -c min1.ncrst -r min2.ncrst

# Stage 3: 0→300 K 加熱 (DNA に弱い拘束)
sander -O \
    -i heat.in -o heat.out \
    -p $PARM -c min2.ncrst -r heat.ncrst -x heat.nc \
    -ref min2.ncrst    # ★ 加熱中の拘束参照は最小化後の構造

# Stage 4: 本番 NPT MD
sander -O \
    -i prod.in -o prod.out \
    -p $PARM -c heat.ncrst -r prod.ncrst -x prod.nc
```

### `-ref` フラグについて (新登場)

拘束を使うときの新しい sander オプション:

| フラグ | 役割 |
|---|---|
| `-c` | 入力座標 (= MD のスタート位置) |
| `-ref` | **拘束の基準座標** (= ここからどれだけズレたかを測る) |

通常は `-c` と `-ref` を同じにすることが多いですが、概念的には別物。

### 所要時間目安

| 段階 | 所要時間 (CPU) |
|---|---|
| min1 | 数十秒 |
| min2 | 数分 |
| heat (20 ps) | 5〜10 分 |
| prod (100 ps) | 30〜60 分 |

合計で **1 時間程度**。本番を 1 ns に伸ばすと **数時間〜半日**。

## 6. 出力の見方

### 段階ごとに何を見るか

| ファイル | チェックポイント |
|---|---|
| `min1.out` | 溶媒のエネルギーが下がる (DNA エネルギーは固定なので変化なし) |
| `min2.out` | 全系エネルギーが負の値で安定 |
| `heat.out` | 温度が 0→300 K に上昇、最終的に 300±5 K で振動 |
| `prod.out` | 温度 300 K、密度 1.0 g/cc、圧力が 1 atm 周辺で振動 |

### 拘束エネルギーを確認

`.out` ファイルの `RESTRAINT` 列が拘束による余分なエネルギー:

```
RESTRAINT  =     12.34
```

加熱の最後でこの値が**小さい (= 残基がほぼ初期位置)** ならうまく拘束が効いている証拠。

## 7. つまずきポイント

| 症状 | 対処 |
|---|---|
| `Could not find restraint mask: RES 1 20` | 拘束ブロック後の `END END` が抜けている可能性 |
| heat 後に密度が異常 | NVT で固定体積中で温度を上げると、密度は溶媒和時の値のまま。NPT 段階 (段階 4) で正しい密度に戻る |
| 加熱が爆発する | 拘束強度を上げてみる (`10.0` → `25.0`) |
| min1 が遅い | 水分子が多いほど時間がかかる。1000 サイクルで終わらなければ `maxcyc=2000` に増やす |
| `-ref` を忘れた | `ntr=1` を使う段階では `-ref` 必須。なしだとエラーまたは挙動がおかしくなる |
| Stage 4 (prod) で温度がおかしい | `irest=1, ntx=7` の組み合わせを確認。違うと heating の状態を引き継げず、再 0 K から始まる |
| 圧力が大きく振動 | 1 atm 周辺で ±100 atm くらい振動するのは正常。平均が 1 atm 付近にあれば OK |

## 8. 用語ミニ辞典

- **位置拘束 (positional restraint)**: 原子を初期位置の近くに留めるバネ
- **段階的平衡化 (staged equilibration)**: 重い系を安全に平衡に導く手順
- **`-ref` フラグ**: 拘束基準座標を指定する sander オプション
